# webgl-radiosity

Implement radiosity by WebGL.

Use a hemicube algorithm for radiosity rendering in WebGL.

## Run

### Install Dependencies

```sh
yarn
# or
# npm install
```

### Display

```sh
yarn global add live-server
live-server
```

## Based

- [Babylon](https://github.com/BabylonJS/Babylon.js)
